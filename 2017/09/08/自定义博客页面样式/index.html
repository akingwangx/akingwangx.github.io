自定义博客页面样式

发表于 2017-09-08

前言

研究了一下午终于会改样式了，现在用的是NEXT主题，感觉蛮顺手的，本篇用来记录如何修改一些样式并生成到博客上。

找到主题文件夹里用于打包CSS输出成CSS样式的main.css

// CSS Style Guide: http://codeguide.co/#css



$scheme    = hexo-config('scheme') ? hexo-config('scheme') : 'Muse';
$variables = base $scheme custom;
$mixins    = base $scheme custom;



// Variables Layer
// --------------------------------------------------
for $variable in $variables
  @import "_variables/" + $variable


// Mixins Layer
// --------------------------------------------------
for $mixin in $mixins
  @import "_mixins/" + $mixin;



// Common Layer
// --------------------------------------------------

// Scaffolding
@import "_common/scaffolding";

// Layout
@import "_common/outline";

// Components
@import "_common/components";


// Schemes Layer
// --------------------------------------------------
@import "_schemes/" + $scheme;



// Custom Layer
// --------------------------------------------------
@import "_custom/custom"; 这句很关键！

这是加载导出自定义设置的代码，可以覆盖掉上面已导出的CSS，如果想换回原来的样式，只需要将_custom/custom.styl里面的代码删掉就可以了。

大概就是这样，修改后用git $hexo s看一下实现了没然后$hexo g -d就OK啦~
