CSS3实现动态扫光文字

发表于 2017-09-21 | 分类于 Testing

思路

浏览网页发现了一个扫光文字，觉得特别好看，就自己思考并尝试了下，跟我想的差不多可以用纯CSS来实现，

在文字表层加一个动态的光，从左向右加一个动画运动，这个光可以用:after或者:before来实现，然后我们对这个伪元素用CSS的渐变gradient进行处理。

代码如下：

.sample{
    background-color: #0E1326;
    padding-top:30px;
    overflow: hidden;
  }


  .blank_text{
    position: relative;
    width:200px;
    margin:20px auto;
    color: #fff;
    line-height: 1;
    font-size: 50px;
    font-size: 0.74074rem;
    text-align: center;
    overflow: hidden;
    font-family: "icomoon";
  }
.blank_text:after{
    width: 300%;
    height: 100%;
    content: "";
    position: absolute;
    top: 0;
    left: 0;
    background: -webkit-gradient(linear, left top, right top, color-stop(0, rgba(15,20,40, 0.7)), color-stop(0.4, rgba(15,20,40, 0.7)), color-stop(0.5, rgba(15,20,40, 0)), color-stop(0.6, rgba(15,20,40, 0.7)), color-stop(1, rgba(15,20,40, 0.7)));
    -webkit-animation: slide ease-in-out 2s infinite; 
}
@-webkit-keyframes slide{
    0%{-webkit-transform:translateX(-66.666%);}
    100%{-webkit-transform:translateX(0);}
